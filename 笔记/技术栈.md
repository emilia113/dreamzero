# DreamZero 技术栈与硬件要求

## 1. 技术栈

| 组件 | 技术 |
|---|---|
| 配置管理 | Hydra |
| 训练框架 | HuggingFace Transformers (Trainer) |
| 分布式训练 | DeepSpeed ZeRO (Stage 2/3) |
| 混合精度 | bfloat16 + TF32 |
| 实验追踪 | WandB |
| 数据加载 | Ray (分布式) + Decord (视频解码) |
| 参数高效微调 | PEFT (LoRA) |

---

## 2. 硬件需求估算

| 训练方式 | GPU 需求 | 显存/卡 | 参考配置 |
|---|---|---|---|
| LoRA 微调 | 4-8x GPU | ~40GB+ | 8x A100/H100, batch=1-4 |
| 全量微调 | 8x GPU | ~80GB | 8x H100, ZeRO-2 + CPU Offload |
| 推理 | 1-2x GPU | ~28GB+ | GB200 或 H100 |

---

## 3. 预训练权重下载

| 权重 | 大小 | 用途 |
|---|---|---|
| Wan2.1-I2V-14B-480P | ~28GB | 扩散骨干底座 |
| umt5-xxl | ~10GB | 文本编码器 tokenizer |
| DreamZero-DROID | ~45GB | DROID 推理权重 |
| DreamZero-AgiBot | ~45GB | AgiBot 微调起点（推荐） |

建议从 DreamZero-AgiBot checkpoint 开始微调，而非从 Wan 原始权重开始，可大幅减少训练时间。
