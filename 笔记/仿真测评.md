# DreamZero 仿真测评

DreamZero 提供两种仿真评测方式：
1. **sim-evals**：简单的 Isaac Sim 仿真 demo，只有 3 个场景，不支持量化成功率
2. **MolmoSpaces**：正式 benchmark，8 个任务，支持自动计算成功率，DreamZero 在排行榜排名第一

两者都采用 **Server-Client 架构**：DreamZero 推理服务器通过 WebSocket 提供动作预测，仿真客户端负责环境交互。

---

## 1. sim-evals（简单测试）

### 简介

- 仓库：`https://github.com/arhanjain/sim-evals`
- 基于 NVIDIA Isaac Sim（IsaacLab）
- **只有 3 个场景**，仅用于简单测试模型是否能跑，不支持量化成功率指标
- 参考 Issue #35：作者明确说 "The droid-sim doesn't support quantifying any success metric"

### 场景

| scene | 任务指令 |
|-------|---------|
| 1 | "put the cube in the bowl" |
| 2 | "pick up the can and put it in the mug" |
| 3 | "put the banana in the bin" |

### 安装

```bash
cd sim-evals
conda create -n sim-evals python=3.11
conda activate sim-evals

# flatdict 需要提前装，避免 build isolation 问题
pip install flatdict==4.0.1
pip install -e .

# 下载仿真资源
uvx hf download owhan/DROID-sim-environments --repo-type dataset --local-dir assets
```

> 注意：sim-evals 依赖 isaaclab/isaacsim，与 websockets 版本可能冲突，需在 pyproject.toml 中将 `websockets>=16.0` 改为 `websockets>=12.0`。

### 使用

**终端 1：启动 DreamZero 推理服务器**

```bash
cd /path/to/dreamzero
python eval_utils/serve_dreamzero_wan22.py \
  --model_path ./checkpoints/你的checkpoint目录 \
  --port 8000
```

**终端 2：运行仿真评测**

```bash
cd sim-evals
python eval_utils/run_sim_eval.py \
  --host localhost \
  --port 8000 \
  --episodes 10 \
  --scene 1 \
  --headless
```

参数说明：
- `--host / --port`：推理服务器地址
- `--episodes`：运行多少个 episode（回合）
- `--scene`：场景编号（1/2/3）
- `--headless`：无头模式（不显示 GUI）

输出视频保存在 `runs/{日期}/{时间}/episode_N.mp4`。

## 2. MolmoSpaces（正式 Benchmark）

### 简介

- 仓库：`https://github.com/allenai/molmospaces`
- Allen AI 开源，230k+ 室内环境，130k+ 物体资源
- **8 个 benchmark 任务**（pick、open、close 等），支持自动计算成功率
- Sim-to-real 相关性极高（R=0.96, ρ=0.98）
- 基于 **MuJoCo**（不需要 Isaac Sim）
- 排行榜：`https://molmospaces.allen.ai/leaderboard`
- DreamZero 在排行榜排名第一（截至 2026/02/27）

### MolmoSpaces 已内置 DreamZero 接入代码

关键文件：
- `molmospaces/molmo_spaces/policy/learned_policy/dreamzero_policy.py` — DreamZero policy 实现
- `molmospaces/molmo_spaces/configs/policy_configs_baselines.py` — DreamZeroPolicyConfig
- `molmospaces/molmo_spaces/evaluation/configs/evaluation_configs.py` — DreamZeroPolicyEvalConfig
- `molmospaces/molmo_spaces/evaluation/eval_main.py` — 评测入口

已处理的对接逻辑：
- 观测转换：MuJoCo camera 图像 resize 到 180x320
- 关节状态映射：`obs["qpos"]["arm"][:7]` → `observation/joint_position`
- Gripper 归一化：`gripper / 0.824033` 映射到 [0,1]
- 动作缓冲：chunk_size=24
- Gripper 二值化：> 0.5 → 255（闭合），否则 0（张开）
- Session 管理：每个 episode 自动 reset

### 安装

```bash
cd molmospaces
conda create -n mlspaces python=3.11
conda activate mlspaces
pip install -e ".[mujoco]"

# 下载 benchmark 资源
export MLSPACES_ASSETS_DIR=/path/to/assets
python -m molmo_spaces.molmo_spaces_constants
```

### 配置

修改 `molmo_spaces/configs/policy_configs_baselines.py` 中的 DreamZeroPolicyConfig：

```python
class DreamZeroPolicyConfig(BasePolicyConfig):
    checkpoint_path: str = "checkpoints/dreamzero"
    remote_config: dict = dict(host="localhost", port=8000)  # 改成你的服务器端口
    grasping_type: str = "binary"
    grasping_threshold: float = 0.5
    chunk_size: int = 24
```

### 使用

**终端 1：启动 DreamZero 推理服务器**

```bash
cd /path/to/dreamzero
python eval_utils/serve_dreamzero_wan22.py \
  --model_path ./checkpoints/你的checkpoint目录 \
  --port 8000
```

多卡推理：
```bash
CUDA_VISIBLE_DEVICES=0,1 python -m torch.distributed.run \
  --standalone --nproc_per_node=2 \
  eval_utils/serve_dreamzero_wan22.py \
  --model_path ./checkpoints/你的checkpoint目录 \
  --port 8000
```

**终端 2：运行 MolmoSpaces 评测**

```bash
cd molmospaces
python molmo_spaces/evaluation/eval_main.py \
  molmo_spaces.evaluation.configs.evaluation_configs:DreamZeroPolicyEvalConfig \
  --benchmark_dir assets/bench/path-to-benchmark.json \
  --task_horizon_steps 500
```

参数说明：
- 第一个参数：评测配置类（DreamZeroPolicyEvalConfig）
- `--benchmark_dir`：benchmark JSON 文件路径
- `--checkpoint_path`：可选，覆盖配置中的 checkpoint 路径
- `--task_horizon_steps`：每个 episode 最大步数
- `--task_horizon_sec`：每个 episode 最大秒数（与 steps 二选一）
- `--num_workers`：并行 worker 数
- `--max_episodes`：最多评测多少个 episode
- `--output_dir`：输出目录
- `--no_wandb`：禁用 wandb 日志

也可以在 Python 代码中调用：

```python
from molmo_spaces.evaluation import run_evaluation

results = run_evaluation(
    eval_config_cls="molmo_spaces.evaluation.configs.evaluation_configs:DreamZeroPolicyEvalConfig",
    benchmark_dir="/path/to/benchmark",
    checkpoint_path="/path/to/checkpoint",
    task_horizon_steps=500,
)
print(f"成功率: {results.success_rate:.1%}")
print(f"成功/总数: {results.success_count}/{results.total_count}")
```

### 输出

- 评测结果保存在 `eval_output/DreamZeroPolicyEvalConfig/{timestamp}/`
- 包含每个 episode 的成功/失败状态
- 可选 wandb 日志（包含视频回放）

## 对比总结

| | sim-evals | MolmoSpaces |
|---|---|---|
| **用途** | 简单 demo 测试 | 正式 benchmark |
| **场景数** | 3 | 8 个任务，230k+ 环境 |
| **成功率指标** | 不支持 | 支持自动计算 |
| **仿真引擎** | Isaac Sim | MuJoCo |
| **DreamZero 接入** | eval_utils/run_sim_eval.py | 已内置 dreamzero_policy.py |
| **安装难度** | 较高（Isaac Sim 依赖重） | 较低（MuJoCo 轻量） |
| **排行榜** | 无 | molmospaces.allen.ai/leaderboard |
| **推荐** | 快速验证能否跑通 | 正式评测用这个 |
